Question-1: What are the different ways to select an element in the DOM?

We can select elements in the DOM using methods like getElementById for IDs, getElementsByClassName for classes, getElementsByTagName for tags, querySelector or querySelectorAll for CSS selectors, and getElementsByName for name attributes. These help us target specific elements on a webpage.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML content inside an element, including tags. innerText returns only the visible text content, ignoring hidden elements and respecting styling. textContent returns all text content, including hidden elements, but ignores HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element instead of adding multiple listeners to individual child elements. Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates or "bubbles up" through its parent elements in the hierarchy. For example, if you click a button, the click event will bubble up through its parent elements, allowing you to handle the event at a higher level.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement. To add it, use appendChild or insertBefore on a parent element. To remove an element, use removeChild on its parent or directly call remove on the element itself.
